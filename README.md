## 👨‍💻 성장을 멈추지 않는 백엔드 개발자

- 저는 **성장을 멈추지 않는 백엔드 개발자**입니다.  
- 기능 구현에만 집중하기보다, **문제의 원인과 구조적인 개선 방법**을 함께 고민하는 개발을 지향합니다.  
- **Spring Boot 기반의 웹 애플리케이션 개발**, **알고리즘 문제 해결**,  
  그리고 **Docker·CI/CD 환경 경험**을 통해 백엔드 기본기를 다져왔습니다.  
- 팀 프로젝트에서는 **책임감 있는 역할 수행과 원활한 소통**을 중요하게 생각하며,  
  **유지보수와 확장성을 고려한 코드 작성**을 목표로 개발해 왔습니다.  
- 앞으로도 꾸준한 학습과 실무 경험을 통해 **신뢰할 수 있는 개발자**로 성장하고자 합니다.
